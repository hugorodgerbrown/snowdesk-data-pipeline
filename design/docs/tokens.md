# Snowdesk design tokens

> Auto-generated by `design/build/style-dictionary.config.mjs`.
> Edit the source JSON under `design/tokens/`, then run `npm run tokens`.

## Why tokens live here

Tokens are sourced from W3C DTCG JSON files so they round-trip cleanly between
code and Figma (via the Tokens Studio plugin). Style Dictionary builds three
consumers: the CSS bundle that Tailwind reads, this Markdown reference, and a
Tokens Studio JSON at `design/figma/tokens.json`.

## Figma bootstrap (one-time)

1. Install Tokens Studio in Figma.
2. Open the plugin → Settings → Token Storage → JSON, paste the contents of
   `design/figma/tokens.json`.
3. Save the imported tokens as a Figma Variables collection.
4. Subsequent designer edits flow back via Tokens Studio export → repo PR.
