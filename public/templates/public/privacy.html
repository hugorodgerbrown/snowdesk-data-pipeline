{% trans "Privacy Policy" %}

{% trans "Snowdesk · snowdesk.info · Effective 1 November 2026 · Version 1.0" %}

{% trans "The short version:" %} {% blocktrans trimmed %} We collect your email address only if you choose to subscribe to bulletin notifications. We don't sell your data. We don't track you across the web. We don't use advertising or analytics cookies. {% endblocktrans %}

{% trans "1. Who we are" %}

{% blocktrans trimmed %} Snowdesk is operated by Henri Tamboral, an individual based in London, UK. Snowdesk is a free service that displays Swiss avalanche bulletins published by the WSL Institute for Snow and Avalanche Research (SLF). There are no employees; all operations are carried out by the sole operator. {% endblocktrans %}

{% blocktrans trimmed %} Questions about this policy? Email: privacy@snowdesk.info {% endblocktrans %}

{% trans "2. What data we collect" %}

{% trans "2a. Subscriptions" %}

{% blocktrans trimmed %} If you subscribe to bulletin email notifications, we collect your email address. That is the only personal data we ever ask for. We collect it only when you actively sign up — we never collect it passively. {% endblocktrans %}

{% trans "2b. Location" %}

{% blocktrans trimmed %} The interactive map can use your browser's optional geolocation to centre the map on your current position. This happens entirely in your browser — your location is never transmitted to or stored by Snowdesk. {% endblocktrans %}

{% trans "2c. Session data" %}

{% blocktrans trimmed %} We use a session cookie that is strictly necessary for the site to function. It contains no personal information and expires when you close your browser. No consent is required for this cookie under UK PECR. {% endblocktrans %}

{% trans "2d. Server logs" %}

{% blocktrans trimmed %} Our server automatically records standard technical information when you visit: your IP address, the page you requested, and the time of the request. We use this only to keep the service running and to investigate errors. Logs are retained for 14 days and then deleted. {% endblocktrans %}

{% trans "3. Why we use your data" %}

{% trans "We process your data under the following legal bases (UK GDPR):" %}

  • {% blocktrans trimmed %} Email address — consent. You gave it to us voluntarily when you subscribed, and you can withdraw it at any time by unsubscribing. {% endblocktrans %}
  • {% blocktrans trimmed %} Session data and server logs — legitimate interest. We need these to keep the service secure and operational. {% endblocktrans %}

{% trans "4. How long we keep your data" %}

  • {% blocktrans trimmed %} Email address — kept for as long as your subscription is active. Deleted within 30 days of unsubscribing or on request. {% endblocktrans %}
  • {% trans "Location — never stored." %}
  • {% trans "Server logs — 14 days." %}

{% trans "5. Who we share your data with" %}

{% trans "We do not sell, rent, or trade your personal data with anyone." %}

{% trans "We use a small number of trusted service providers to operate Snowdesk:" %}

  • {% blocktrans trimmed %} Render (render.com) — hosts the Snowdesk servers. Your IP address and request data pass through their infrastructure. {% endblocktrans %}
  • {% blocktrans trimmed %} Resend (resend.com) — sends your bulletin notification emails. They receive your email address for this purpose only. {% endblocktrans %}
  • {% blocktrans trimmed %} Open-Meteo (open-meteo.com) — provides weather data displayed on bulletin pages. No personal data is shared with them. {% endblocktrans %}
  • {% blocktrans trimmed %} SLF / WSL — provides the avalanche bulletin data. No personal data is shared with them. {% endblocktrans %}

{% blocktrans trimmed %} All providers are contractually required to handle your data in compliance with UK GDPR. We do not use any advertising networks or analytics platforms. {% endblocktrans %}

{% trans "6. Your rights" %}

{% trans "Under UK GDPR, you have the right to:" %}

  • {% trans "Access the personal data we hold about you" %}
  • {% trans "Correct inaccurate data" %}
  • {% trans "Request deletion of your data (“right to be forgotten”)" %}
  • {% trans "Object to processing" %}
  • {% trans "Withdraw consent at any time" %}

{% blocktrans trimmed %} To exercise any of these rights, email privacy@snowdesk.info. We will respond within 30 days. {% endblocktrans %}

{% blocktrans trimmed %} You also have the right to lodge a complaint with the UK Information Commissioner's Office (ICO) at ico.org.uk. {% endblocktrans %}

{% trans "7. Children" %}

{% blocktrans trimmed %} Snowdesk is a publicly accessible information service. We do not knowingly collect personal data from children under 13. If you believe a child under 13 has subscribed, please contact us at privacy@snowdesk.info and we will delete the data without delay. {% endblocktrans %}

{% trans "8. Cookies" %}

{% blocktrans trimmed %} Snowdesk uses one strictly necessary session cookie. It is set automatically when you visit the site, contains no personal information, and expires when you close your browser. Because it is strictly necessary, no consent banner is required under UK PECR. We do not use tracking, analytics, or advertising cookies of any kind. {% endblocktrans %}

{% trans "9. Changes to this policy" %}

{% blocktrans trimmed %} If we make material changes to this policy, we will update the effective date at the top and notify subscribers by email. Continued use of Snowdesk after changes constitutes acceptance. {% endblocktrans %}
