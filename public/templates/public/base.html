{% comment %} public/templates/public/base.html — Canonical layout for the public site. Pulled out of bulletin.html (the "most complete, use as base" page flagged in SNOW-9). Every public page except /map/ extends this template; the map page owns its full viewport and keeps its own doctype for that reason. Subscription pages also extend this file indirectly via subscriptions/templates/subscriptions/base.html. Blocks ------ title : contents. Default "Snowdesk". meta_description: <meta name="description"> content. Empty by default. favicon : the <link rel="icon"> tag. Default points at the plain favicon.svg; bulletin.html overrides with a level-tinted variant matching the page's danger rating. extra_head : page-specific <head> content (stylesheets, scripts, embedded JSON). Loaded AFTER output.css so page-level overrides win. body_class : Tailwind classes on <body>. Default gives background, text colour, font, min-h-screen, and flex-col so pages with a centred hero (home.html) can rely on flex-1 to fill remaining vertical space. nav : top navigation. Default = the plain wordmark variant; pages that want a back-chevron override this block. content : the page body. site_footer : the global site footer. Default = the small SLF attribution + Terms link partial introduced in SNOW-30 to satisfy the SLF CC BY 4.0 licence. Pages with a richer per-page footer (bulletin.html) still inherit this line below the rich one — the rich footer is page-content; this is the site licence line. Override with {% block site_footer %}{% endblock %} only if a page needs to reposition the licence line (e.g. into a floating overlay), not to remove it. extra_js : scripts loaded at end of <body>.
