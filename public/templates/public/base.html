{% comment %} public/templates/public/base.html — Canonical layout for the public site. Pulled out of bulletin.html (the "most complete, use as base" page flagged in SNOW-9). Every public page except /map/ extends this template; the map page owns its full viewport and keeps its own doctype for that reason. Subscription pages also extend this file indirectly via subscriptions/templates/subscriptions/base.html. Blocks ------ title : contents. Default "Snowdesk". meta_description: <meta name="description"> content. Empty by default. favicon : the <link rel="icon"> tag. Default points at the plain favicon.svg; bulletin.html overrides with a level-tinted variant matching the page's danger rating. extra_head : page-specific <head> content (stylesheets, scripts, embedded JSON). Loaded AFTER output.css so page-level overrides win. body_class : Tailwind classes on <body>. Default gives background, text colour, font, min-h-screen, and flex-col so pages with a centred hero (home.html) can rely on flex-1 to fill remaining vertical space. nav : top navigation. Default = the plain wordmark variant; pages that want a back-chevron override this block. content : the page body. extra_js : scripts loaded at end of <body>. Used by subscriptions/base.html to inject HTMX. No global footer — the canonical bulletin page owns a rich per-page footer rather than a site-wide one, and home/map/random/season all omit a footer too.
