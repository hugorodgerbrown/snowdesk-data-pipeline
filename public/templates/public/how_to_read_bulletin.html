{% trans "How to read the avalanche bulletin" %}

{% blocktrans trimmed %} Published daily by the WSL Institute for Snow and Avalanche Research (SLF), the bulletin covers the Swiss Alps and Jura. It is an informed forecast — not a guarantee — and cannot replace an on-site assessment. This guide explains how it is structured and what each field means. {% endblocktrans %}

{# ------------------------------------------------------------------ #} {# Danger level #} {# ------------------------------------------------------------------ #}

{% trans "The danger level" %}

{% trans "The bulletin uses the five-point European avalanche danger scale." %}

{% trans "Level" %} {% trans "Name" %} {% trans "Characteristics" %}
1 {% trans "Low" %} {% trans "Generally favourable. Avalanches only in isolated cases on extremely steep slopes." %}
2 {% trans "Moderate" %} {% trans "Mostly favourable. Avalanches possible on steep slopes with the indicated aspects and elevation." %}
3 {% trans "Considerable" %} {% trans "Critical. Avalanches can be easily triggered on steep slopes; natural avalanches and remote triggering possible." %}
4 {% trans "High" %} {% trans "Very critical. Natural avalanches likely; easily triggered on many steep slopes." %}
5 {% trans "Very high" %} {% trans "Extraordinary. Numerous very large natural avalanches expected." %}

{% blocktrans trimmed %} Danger increases disproportionately — the jump from level 2 to 3 is far more significant than the numbers suggest. Around 50% of avalanche fatalities occur at level 3, 30% at level 2. {% endblocktrans %}

{% blocktrans trimmed %} In Swiss bulletin data covering October 2025 to May 2026, the distribution across 2,159 bulletins was: low 20%, moderate 54%, considerable 35%, high 5%, very high <1%. (Some bulletins carry two danger levels for different times of day; these percentages reflect the all-day rating.) {% endblocktrans %}

{# ------------------------------------------------------------------ #} {# Subdivisions #} {# ------------------------------------------------------------------ #}

{% trans "Subdivisions (–, =, +)" %}

{% blocktrans trimmed %} When there is a moderate or higher danger of dry-snow avalanches (level 2 and above), the SLF subdivides each level into three positions: {% endblocktrans %}

  • {% trans "– (minus)" %} {% trans "— towards the bottom end of the level" %}
  • {% trans "= (neutral)" %} {% trans "— more or less in the middle" %}
  • {% trans "+ (plus)" %} {% trans "— towards the top end of the level" %}

{% blocktrans trimmed %} This reflects the fact that avalanche danger is a continuous spectrum, not a step function. A rating of 3– sits just above the level 2/3 boundary; 3+ is just below level 4. {% endblocktrans %}

{% blocktrans trimmed %} Subdivision only applies to dry-snow danger. It is never assigned to: {% endblocktrans %}

  • {% blocktrans trimmed %}Level 1 (low) — confirmed across all 2,159 bulletins in the dataset; not a single low rating carries a subdivision{% endblocktrans %}
  • {% trans "Wet-snow hazards — wet snow and gliding snow are not assessed with subdivisions" %}
  • {% trans "The afternoon period — the later danger rating is always wet, so subdivision never appears there" %}

{% trans 'A missing subdivision means "not applicable", not "neutral".' %}

{# ------------------------------------------------------------------ #} {# Hazard categories #} {# ------------------------------------------------------------------ #}

{% trans "Hazard categories: dry and wet" %}

{% trans "The bulletin assesses dry and wet avalanche hazards separately. These are fundamentally different in cause, location, and behaviour." %}

{% trans "Dry avalanches" %}

{% blocktrans trimmed %} Caused by structural weaknesses in the snowpack — new snow loading, wind-deposited slabs, or persistent buried weak layers. Dry hazard is always present throughout the day; it does not emerge or disappear with temperature. In the dataset, dry hazards appear exclusively as an all-day rating. {% endblocktrans %}

{% blocktrans trimmed %} Dry problems always carry a specific lower elevation bound (the altitude above which the hazard applies) and one or more aspects (compass directions of affected slopes). Dry hazard lives above a threshold, not below one. {% endblocktrans %}

{% trans "Problem type" %} {% trans "Description" %}
{% trans "New snow" %} {% trans "Recent snowfall loading the existing snowpack. Typically widespread across all aspects." %}
{% trans "Wind slab" %} {% trans "Wind-deposited snow on leeward slopes, gullies, and bowls. Localised but often hard to spot when buried." %}
{% trans "Persistent weak layers" %} {% blocktrans trimmed %} Buried weak layers — faceted crystals, depth hoar, surface hoar — that can remain reactive for weeks. Dangerous because avalanche-prone locations are barely recognisable even to trained eyes. The dominant winter hazard. {% endblocktrans %}
{% trans "No distinct avalanche problem" %} {% trans "No clear dry hazard pattern. Not equivalent to no danger — any avalanche type remains possible. Typically used at level 1." %}

{% trans "Wet avalanches" %}

{% blocktrans trimmed %} Caused by liquid water weakening the snowpack, whether from melt (solar radiation, warm air) or rain. Wet hazard can be present all day or emerge progressively as temperatures rise — the only hazard type that is time-dependent. {% endblocktrans %}

{% blocktrans trimmed %} Wet problems are spatially more diffuse than dry ones. At moderate wet danger, most cases specify aspects with an upper elevation bound (the hazard exists below the freeze-thaw or rain/snow line). At higher danger or when the hazard is limited to the afternoon, spatial constraints are often absent — the hazard is treated as region-wide. {% endblocktrans %}

{% trans "Problem type" %} {% trans "Description" %}
{% trans "Wet snow" %} {% blocktrans trimmed %} Snowpack weakened by melt or rain. Mainly natural avalanches. Solar radiation and aspect determine which slopes are affected first. Timing is critical — east-facing slopes warm earlier than west-facing ones. {% endblocktrans %}
{% trans "Gliding snow" %} {% blocktrans trimmed %} The entire snowpack slides on a smooth substrate (grass, rock). Almost exclusively natural; human triggering is virtually impossible. Glide cracks are a warning sign but release timing is unpredictable. Can occur at any time of day, including overnight. {% endblocktrans %}
{# ------------------------------------------------------------------ #} {# Elevation and aspect #} {# ------------------------------------------------------------------ #}

{% trans "Elevation and aspect" %}

{% trans "Elevation" %} {% blocktrans trimmed %} is given as a lower bound for dry hazards ("above 2400 m") or an upper bound for wet hazards ("below 2400 m"). These are approximate — conditions change gradually across the boundary, not sharply. A ±200 m margin is reasonable. {% endblocktrans %}

{% trans "Aspects" %} {% blocktrans trimmed %} are compass directions of affected slopes. The convention is: a north-facing slope falls to the north — you stand at the top, facing north, and the slope drops away below you. North-facing slopes receive no direct solar radiation in mid-winter; south-facing slopes receive regular sunshine year-round. {% endblocktrans %}

{% blocktrans trimmed %} When the bulletin cites specific aspects and an elevation, the stated danger level applies to slopes satisfying both criteria. A common rule of thumb for slopes outside the stated zone: assume one danger level lower. This is a guideline, not a guarantee. {% endblocktrans %}

{% trans "When no aspects or elevation are given for a wet hazard, the danger is region-wide and not spatially constrained." %}

{# ------------------------------------------------------------------ #} {# How the day evolves #} {# ------------------------------------------------------------------ #}

{% trans "How the day evolves" %}

{% blocktrans trimmed %} Most bulletins describe a single, stable hazard picture for the whole day. In about 8% of bulletins, a second danger level applies later in the day — exclusively a wet-snow hazard emerging as temperatures rise. {% endblocktrans %}

{% blocktrans trimmed %} All bulletins where the afternoon differs from the morning involve wet snow or gliding snow. Dry hazards never appear as a later-in-the-day entry. This reflects the physics: dry snowpack problems are weather-independent within a single day, while wet hazards are driven by daytime heating. {% endblocktrans %}

{% blocktrans trimmed %} The bulletin reflects this with two danger ratings when the day evolves. The morning rating shows the all-day dry baseline; the afternoon rating shows the wet hazard that develops. The stated danger level is normally based on the morning situation. {% endblocktrans %}

{% trans "Typical patterns" %}

{% trans "Pattern" %} {% trans "Frequency" %} {% trans "Description" %}
{% trans "Dry only, all day" %} 70% {% trans "A single dry hazard with no wet component." %}
{% trans "Dry and wet simultaneously" %} 16% {% trans "Both dry (above an elevation) and wet (below it) are active all day — often after rain at lower elevations with new snow higher up." %}
{% trans "Dry all day, wet emerging later" %} 8% {% trans "Morning is dominated by dry hazard; afternoon brings wet snow as temperatures rise." %}
{% trans "Wet only, all day" %} 5% {% trans "No dry hazard — typically low-elevation regions or late spring." %}

{% blocktrans trimmed %} When the afternoon brings a higher danger level than the morning, timing is critical: aim to complete exposed terrain before the snow softens. On east-facing slopes this may mean before midday; on west-facing slopes it may be mid-afternoon. The bulletin cannot pin down the exact transition time — it depends on aspect, elevation, and conditions on the day. {% endblocktrans %}

{# ------------------------------------------------------------------ #} {# Multiple problem types #} {# ------------------------------------------------------------------ #}

{% trans "When multiple problem types appear together" %}

{% blocktrans trimmed %} A single hazard block may cite two problem types — for example, wind slab and persistent weak layers together. This means both contribute to the danger in the same area, at the same elevation, on the same aspects. When two problem types are grouped together they always share identical spatial constraints and description text. They are two labels for one danger zone, not two independent hazards with different locations. {% endblocktrans %}

{% blocktrans trimmed %} Multiple problem types within one hazard block occur in around 47% of bulletins. The most common pairing is wind slab and persistent weak layers. {% endblocktrans %}

{# ------------------------------------------------------------------ #} {# Narrative sections #} {# ------------------------------------------------------------------ #}

{% trans "The narrative sections" %}

{% trans "Each bulletin includes longer text in four sections, updated once daily in the evening:" %}

{% trans "Snowpack" %}
{% trans "Describes the layering and stability of the snowpack across the region. Explains why the danger is as it is. Mentions observed avalanche activity. Most directly useful for planning." %}
{% trans "Weather forecast" %}
{% trans "Covers the next 24 hours: fresh snow amounts, temperature, wind. The danger assessment is built on this forecast; if actual weather deviates, the danger may differ." %}
{% trans "Weather review" %}
{% trans "Summarises what actually happened in the preceding period. Provides context for why the snowpack is in its current state." %}
{% trans "Outlook (tendency)" %}
{% trans "Indicative assessment for the two days after the bulletin's validity period. Less detailed but useful for multi-day planning." %}
{# ------------------------------------------------------------------ #} {# Limitations #} {# ------------------------------------------------------------------ #}

{% trans "Limitations" %}

{% blocktrans trimmed %} The bulletin is a regional forecast. It cannot assess individual slopes — it can only indicate which types of terrain (aspect, elevation, steepness) carry the indicated danger. Local conditions always require on-site assessment. {% endblocktrans %}

{% blocktrans trimmed %} Boundaries between regions, elevation bands, and aspects are gradual, not sharp. A slope near the boundary of the stated zone may sit in a transition area where either the more or less favourable situation applies. {% endblocktrans %}

{% trans "The bulletin applies exclusively to unsecured terrain away from prepared pistes and marked ski runs." %}
