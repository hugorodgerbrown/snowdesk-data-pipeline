{% comment %} public/templates/public/_region_peek.html — Peek-state body for the region sheet on /map/. Rendered server-side by ``public.api.region_summary`` and injected into the sheet's peek container by ``static/js/map.js``. Holds only what's visible at the peek snap-point: region name, today's danger chip, and a one-line subtitle composed of one small coloured chip + trait title per render-model trait, joined with a middle-dot separator. The per-trait chips share the same EAWS palette as the main danger tile in the header, so users can see at a glance which trait carries which level — the header chip shows only the *highest* level overall. Expects the following variables in scope: region — the Region instance (for ``region.name``). rm — the bulletin's render_model dict. {% endcomment %} {% load i18n snowdesk_time %}
