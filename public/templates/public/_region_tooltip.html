{% comment %} public/templates/public/_region_tooltip.html — Tooltip body for the region MapLibre Popup on /map/. Server-rendered by ``public.api.region_summary`` and injected into a ``maplibregl.Popup`` via ``popup.setHTML(data.html)`` in ``static/js/map.js``. The popup is anchored to the click point (or the region's bbox centre for deep-link / resort-pin paths). Content: 1. Header row — danger-tile chip (when a bulletin exists) or default Snowdesk icon (when no RegionDayRating row exists), and region name. The chip is presentational only; the bulletin CTA below is the sole link out of the popup. 2. Geographic breadcrumb — three levels in English: country › major (L1) › sub-region (L2). Uses ``name_en`` on MajorRegion / SubRegion, falling back to ``name_native`` for any future entry that lacks an English variant. 3. Bulletin CTA — when a bulletin exists, a button-styled link to open it for the displayed date. When no bulletin is available, a muted "No bulletin available for…" label is shown instead. Context variables expected: region — MicroRegion instance, with subregion__major select_related. day_rating — RegionDayRating instance for the displayed date, or None. bulletin_url — Absolute URL for the day's bulletin (dated or evergreen). country_name — Human-readable country name (e.g. "Switzerland"). target_date — datetime.date for the displayed day. {% endcomment %} {% load i18n snowdesk_time static %}
