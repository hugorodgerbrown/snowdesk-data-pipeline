{% comment %} public/templates/_components/partials/_swatches.html — Colour swatch grid. Rendered for any FoundationCategory with kind="swatches". Each swatch is a coloured square (background = ``var()``) with the token name + human label + value caption. The same markup picks up the *light* value when rendered outside a ``.dark`` ancestor and the *dark* value when rendered inside one — the grid is theme-agnostic. Layout: defaults to a responsive 2→3→4 column grid. Categories whose tokens have a meaningful row/column structure (e.g. EAWS, where the five danger levels repeat across multiple stops) override this via ``swatch_columns`` so the columns line up by level. Context: category : FoundationCategory whose tokens are rendered. {% endcomment %}
