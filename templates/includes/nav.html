{% comment %} templates/includes/nav.html — Persistent top navigation bar. Renders the Snowdesk wordmark plus an optional back link, an optional Season trigger button, and a right-side auth/admin area. Right-side auth area -------------------- Authenticated subscriber: An avatar button (first letter of email) opens a dropdown showing up to three subscribed region links, a "Manage alerts" link, and a "Sign out" button. Region links are provided by the nav_subscriptions context processor (subscriptions/context_processors.py). Not authenticated: A "Sign in" link routes to /subscribe/sign-in/. Staff overlay (separate from subscriber auth): A cog-icon button opens an admin dropdown: Component library, Edit map, Django admin. Rendered in addition to the subscriber avatar — a staff user who is also a subscriber sees both. Parameters ---------- back_url : str, optional URL for a back-chevron link. Omit on pages where "back" has no obvious target (home, map). When present, a thin vertical divider separates the chevron from the wordmark. back_label : str, optional Label shown next to the chevron. Defaults to "Back" — prefer a destination-specific label ("Map", "Season") where possible so the user knows where the arrow leads. season_trigger : bool, optional When truthy, a labelled "SEASON" button is rendered right-aligned.
