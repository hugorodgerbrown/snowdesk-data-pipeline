{% load i18n static %} {% comment %} templates/includes/bulletin_header.html — unified bulletin header panel. Collapses the previous masthead + weather band into a single panel carrying location wayfinding (region + sub-region), date, and weather (hero icon + condition label + sunrise/sunset). The bucket colour (SNOW-98) remains visible for the at-a-glance day/night signal; a ``bg-black/35`` overlay sits between the bucket colour and the content so white text reads against any bucket × time-of-day combination. The date picker for the page lives in the page nav as the "🗓 SEASON" sheet trigger (SNOW-117), so this header carries no calendar affordance. Required context (added by ``public.views.bulletin_detail``): page_date — datetime.date for the focal day. region_name — display name for the H1. subregion_name — parent EAWS L2 sub-region (English where SLF publishes one, native otherwise), or empty. weather_display — dict from ``build_weather_display()``, or None. Shape: ``{weather, bucket, is_day, time_of_day, sunrise_local, sunset_local, icon_bucket, condition_label, icon_filename}``. When ``weather_display`` is None the panel still renders so the page chrome stays consistent — the hero icon is omitted, the metadata strip drops the weather/sunrise lines, and the panel falls back to a neutral dark token via ``data-weather-bucket="none"``. {% endcomment %}
