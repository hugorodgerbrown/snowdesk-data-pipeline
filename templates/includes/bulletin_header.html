{% load i18n static %} {% comment %} templates/includes/bulletin_header.html — unified bulletin header panel. Collapses the previous masthead + weather band into a single panel carrying location wayfinding (region + sub-region), date (with calendar trigger), and weather (hero icon + condition label + sunrise/sunset). The bucket colour (SNOW-98) remains visible for the at-a-glance day/night signal; a ``bg-black/35`` overlay sits between the bucket colour and the content so white text reads against any bucket × time-of-day combination. The calendar glyph is the only tappable affordance — it opens the same calendar drawer the masthead used to surface from its eyebrow. Required context (added by ``public.views.bulletin_detail``): page_date — datetime.date for the focal day. region_name — display name for the H1. subregion_name — parent EAWS L2 sub-region (English where SLF publishes one, native otherwise), or empty. calendar_partial_url — HTMX endpoint for the calendar fragment. weather_display — dict from ``build_weather_display()``, or None. Shape: ``{weather, bucket, is_day, time_of_day, sunrise_local, sunset_local, icon_bucket, condition_label, icon_filename}``. When ``weather_display`` is None the panel still renders so the page chrome stays consistent — the hero icon is omitted, the metadata strip drops the weather/sunrise lines, and the panel falls back to a neutral dark token via ``data-weather-bucket="none"``. {% endcomment %}
