{% load i18n static %} {% comment %} templates/includes/bulletin_weather_header.html — weather-driven page header (SNOW-98, SNOW-100). Renders a graphic band whose background changes with the current weather bucket (clear, partly_cloudy, cloudy, fog, rain, snow, thunder) and time of day (day, night). A hero-sized Meteocons icon and condition label sit inside the left side of the band (SNOW-100). Required context (added by public.views.bulletin_detail): weather_display — dict from bulletins.services.weather_display, or None. Shape: {weather, bucket, is_day, time_of_day, sunrise_local, sunset_local, icon_bucket, condition_label, icon_filename}. When weather_display is None (no snapshot for this region/day, or this template is used by a non-bulletin page), nothing is rendered. Debug overlay: while the visual design is iterating, the band shows the WMO code, bucket, sunrise/sunset (local wall-clock at the bulletin region), and the resolved time-of-day. Strip the inner block once the design lands. Accessibility: the band is currently decorative + debug-only and is hidden from the accessibility tree.
